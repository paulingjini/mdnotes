### Toast Notifications

Toast notifications appear in the top-right corner (or full-width on mobile) and auto-dismiss after 3 seconds.

**JavaScript API:**
```javascript
// Success notification (green)
Toast.success('File saved successfully!');

// Error notification (red)
Toast.error('Export failed');

// Warning notification (orange)
Toast.warning('Connection lost');

// Info notification (blue)
Toast.info('Processing...');

// Custom duration (in milliseconds)
Toast.show('Custom message', 'success', 5000); // 5 seconds

// Dismiss all toasts
Toast.dismissAll();
```

**Features:**
- Auto-dismiss with progress bar
- Close button on each toast
- Stack multiple toasts
- Glassmorphism design
- Slide-in/out animations
- Mobile responsive

### Project Architecture

MDNotes Pro uses a **modular architecture**:

- **Design System First** - 150+ CSS variables for consistency
- **Component-based** - Reusable UI components
- **Module Pattern** - ES6 modules for organization
- **Single File Output** - Bundled for easy deployment

### Adding a New Feature

1. Create module in `src/js/modules/` or `src/js/extensions/`
2. Import in `src/js/app.js`
3. Add to `build/build.js` modules list
4. Add UI in `src/html/template.html`
5. Style in appropriate CSS file
6. Build: `npm run build`
7. Test: Open `dist/index.html`

## ğŸ› Troubleshooting

### App won't load
- Check browser console for errors
- Ensure JavaScript is enabled
- Try clearing browser cache
- Use a modern browser (Chrome, Firefox, Safari, Edge)

### Files not saving
- Check localStorage quota (5-10MB)
- Ensure browser allows localStorage
- Try clearing old files
- Check browser privacy settings

### Export not working
- Ensure popup blocker is disabled
- Check browser download settings
- Verify file size is reasonable (< 10MB)
- Try a different export format

### Slow performance
- Clear old files and archives
- Reduce document size
- Disable unnecessary features
- Close other browser tabs
- Try a lighter theme
